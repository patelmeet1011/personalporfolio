Project: Excel Visualization Web App ("Excel Viz Pro")

Overview

Developed "Excel Viz Pro," a user-friendly web application built with Python and Streamlit. This tool empowers users to upload Excel files (CSV or XLSX format) and instantly generate dynamic, interactive visualizations such as bar charts, line charts, scatter plots, and more. The application aims to simplify the process of exploratory data analysis directly from common spreadsheet formats, making data insights more accessible.

Key Features & Functionality

  • File Upload: Supports CSV and XLSX file formats.
  • Dynamic Visualization: Automatically generates various interactive charts (bar, line, scatter, etc.) based on uploaded data using Plotly.
  • User Interaction: Allows users to potentially select columns for plotting and customize chart types.
  • Report Download: Enables downloading visualizations or summary reports.
  • Intuitive Interface: Built with Streamlit for a simple and responsive user experience.
  • Efficient Data Handling: Uses Pandas and NumPy for backend data manipulation.

The project focused on implementing a user-centric design to provide actionable insights through an easy-to-use interface, streamlining basic data exploration tasks.
